Application Python de gestion de tâches

Ce projet a été développé dans le cadre de ma formation en BUT Informatique. L'objectif était de créer une application permettant de gérer efficacement ses tâches quotidiennes.

Fonctionnalités principales :

  • Création, modification et suppression de tâches
  • Assignation de priorités (haute, moyenne, basse)
  • Définition de dates d'échéance
  • Filtrage et tri des tâches selon différents critères
  • Interface utilisateur intuitive et réactive
  • Sauvegarde des données en base de données SQLite

Technologies utilisées : Python, Tkinter pour l'interface graphique, SQLite pour la base de données.

Compétences acquises : Programmation orientée objet, gestion de base de données, conception d'interface utilisateur, gestion des événements.

Automatisation de tests logiciels

Ce projet a été réalisé dans le cadre d'un stage en entreprise. Il avait pour objectif d'améliorer le processus de test d'une application web en automatisant les tests répétitifs.
